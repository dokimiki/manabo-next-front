# manabo-next-front

## 環境構築

1. 必要なツールのインストール
    - bun
    - git
    - Github Desktop
    - Visual Studio Code
    - Google Chrome

2. リポジトリのクローン

    ```bash
    git clone https://github.com/dokimiki/manabo-next-front.git
    ```

3. パッケージのインストール

    ```bash
    bun i
    ```
